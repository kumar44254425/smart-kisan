# 🌾 Smart Kisan

A web-based platform for farmers providing smart agricultural solutions.

## Features
- 🌦 Weather Forecast (State-wise)
- 📊 Weekly Weather Graph
- 📱 SMS Alerts
- 🌾 Crop Suggestions
- 🛒 Buy Seeds & Fertilizers
- 📦 Order Tracking
- 🏛 Government Scheme Eligibility

## Tech Stack
- Java Servlets
- JSP
- MySQL
- Apache Tomcat
